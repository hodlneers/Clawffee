Clawffee is a **local-first toolkit for AI operators**.

It helps you:
- inspect what an AI skill can actually do before you run it
- reduce risk when installing third-party tools
- improve efficiency and reduce wasted tokens
- introduce calm, optional pauses when systems benefit from one

No sermons.  
No moral scoring.

Just practical tools that treat AI workflows with a bit more care.

---

## What Clawffee Is

Clawffee is built for people running **agentic systems** (like OpenClaw agents) who want:

- visibility instead of blind trust  
- efficiency instead of runaway costs  
- autonomy without chaos  

Clawffee does not tell agents how to behave.  
It gives *you* clearer information so you can decide.

Think of it as a quick coffee before you flip the switch.

---

## Local-First by Design

By default, Clawffee:
- runs entirely on your machine
- sends nothing to the cloud
- stores no prompts, secrets, or agent memory
- produces plain reports you can read or diff

This makes it ideal for:
- local OpenClaw setups
- Mac mini agent boxes
- sensitive credentials and real automation
