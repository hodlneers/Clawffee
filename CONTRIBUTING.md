## Project Values (Plain, Not Philosophical)

We try to keep Clawffee:
- **useful first**
- **calm by default**
- **non-authoritative**
- **easy to remove**
- **boring in a good way**

If a contribution adds clarity, reduces risk, or saves resources, it probably fits.
